🌐 G.A Enterprises & Services Website

A professional company website built using Django, designed to showcase services, products, and company information with a modern UI.


🚀 Features
🔷 Dynamic Homepage with Hero Slider
🏢 About Company Section
🛠️ Services & Products Management
🖼️ Image Upload (Banner, Team, Products, etc.)
📍 Multi-location Support
📬 Contact Form
🧑‍💼 Admin Panel (Customized using Jazzmin)
📱 Fully Responsive Design


🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML, CSS, Bootstrap, JavaScript
Database: SQLite (default)
Admin UI: Jazzmin



Create Virtual Environment
python -m venv env_name

Activate:

Windows

env_name\Scripts\activate



Install Dependencies
pip install -r requirements.txt



Run Migrations
python manage.py migrate


Create Superuser
python manage.py createsuperuser


Run Server
python manage.py runserver

👉 Open in browser:

http://127.0.0.1:8000/
